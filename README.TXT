PROJECT TITLE:Lucky Vending Machine

VERSION or DATE: 07/09/2017
HOW TO START THIS PROJECT: By calling the startGame() method of Game class
AUTHORS: Prashanth

USER INSTRUCTIONS:
1) Create an object of the Game class and run the startGame() method.
2) Then the game menu is displayed on the screen. 
3) First the user needs to set up a new player by selecting option (1) from the game menu , then guess a number between 1 to 5 by selecting option (2) in the game menu.
4) If the player's guess matches with the random number generated by the LuckyVendingMachine, then the player wins a prize.
5) The player can view the prizes won, the total worth of the prizes and also the total amount he/she spent on the game by selecting option (3) from the game menu.
6) Finally, the option (5) can be selected to exit the game.
